# FindaBrew
---
A simple web app that displays a table of breweries with some of their information using the openbrewerydb.

## Technologies Used
---
+ HTML5
+ CSS3
+ JavaScript
+ jQuery
+ Google Fonts
+ fontAwesome
+ openbrewerydb
