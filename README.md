# FindaBrew
---
A web app that displays cards of breweries with their information using the openbrewerydb.

## Technologies Used
---
+ HTML5
+ CSS3
+ JavaScript
+ jQuery
+ Google Fonts
+ fontAwesome
+ openbrewerydb
+ clearbit
